🧠 KnownAgain

AI Memory Companion for Dementia Care

“Imagine waking up and not recognizing your own daughter.”
KnownAgain helps restore familiarity through real-time AI recognition and memory recall.

⸻

🚨 The Problem

Dementia and cognitive decline cause:
	•	Inability to recognize familiar faces
	•	Anxiety during social interaction
	•	Loss of conversational continuity
	•	Emotional distress for patients and families

Patients may speak to someone today…
and forget that interaction tomorrow.

⸻

💡 Our Solution

KnownAgain is an AI-powered memory companion that:

✅ Recognizes familiar faces in real-time
✅ Displays relationship context instantly
✅ Records and summarizes conversations
✅ Extracts meaningful topics
✅ Generates memory insights
✅ Organizes conversations in a calendar view

It acts as a cognitive support layer — restoring familiarity and reducing anxiety.

⸻

🎥 Demo Flow
	1.	Webcam detects visitor
	2.	Face recognized
	3.	Relationship displayed
	4.	“On March 2nd, you talked about your grandson’s exam.”
	5.	Conversation recorded
	6.	Smart summary + keyword extraction
	7.	Stored in structured memory timeline

🚀 How to Run

1️⃣ Clone Repository

git clone "url"
cd knownagain

2️⃣ Create Virtual Environment

python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows

3️⃣ Install Dependencies

pip install flask
pip install opencv-python
pip install face-recognition

4️⃣ Run App

python app.py

Open:

http://127.0.0.1:5000
